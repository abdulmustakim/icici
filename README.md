# Amazon Pay ICICI Credit Card Animated Landing Page

This is a safe informational/demo landing page for the Amazon Pay ICICI Bank Credit Card.

## Files
- `index.html` — main website
- `style.css` — full responsive animated design
- `script.js` — menu, scroll reveal, 3D card tilt, calculator, particles

## Safety Notes
- Do not collect PAN, OTP, Aadhaar, CVV, card number, or banking password.
- Keep the disclaimer visible.
- Use only official Amazon/ICICI pages for the apply button.
- Benefits and reward rates can change. Always verify official terms before publishing.

## How to Run
Open `index.html` in Chrome or any browser.

## How to Deploy
Upload the folder to GitHub Pages, Netlify, Vercel, or any static hosting provider.
